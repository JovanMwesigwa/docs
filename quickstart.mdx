## Build your first payment integration

This quickstart guide will walk you through setting up FundKit and processing your first payment in just a few minutes.

<Info>
  **Prerequisites:** Node.js 16+ and npm/yarn/pnpm/bun installed on your machine.
</Info>

## Step 1: Install FundKit

Install the core package and your chosen providers:

<CodeGroup>

```bash npm
npm install @fundkit/core @fundkit/honeycoin @fundkit/easypay
```

```bash yarn
yarn add @fundkit/core @fundkit/honeycoin @fundkit/easypay
```

```bash pnpm
pnpm add @fundkit/core @fundkit/honeycoin @fundkit/easypay
```

```bash bun
bun add @fundkit/core @fundkit/honeycoin @fundkit/easypay
```

</CodeGroup>

## Step 2: Get API Keys

<AccordionGroup>
  <Accordion icon="flask" title="Sandbox Keys (Recommended for testing)">
    For testing, you can use FundKit's unified sandbox:
    
    ```bash
    FUNDKIT_API_KEY=fk_test_your_sandbox_key_here
    ```
    
    <Tip>Sandbox keys let you test immediately without provider approval!</Tip>
  </Accordion>
  
  <Accordion icon="key" title="Provider Keys (For production)">
    Get API keys from your chosen providers:
    
    **HoneyCoin:**
    - Sign up at [honeycoin.io](https://honeycoin.io)
    - Get your API key and public key from the dashboard
    
    **EasyPay:**
    - Register at [easypay.ug](https://easypay.ug) 
    - Obtain your client ID and API secret
  </Accordion>
</AccordionGroup>

## Step 3: Initialize the Payment Client

Create a new file `payment.js` (or `payment.ts` for TypeScript):

<CodeGroup>

```typescript TypeScript
import { PaymentClient } from '@fundkit/core';
import { HoneyCoin } from '@fundkit/honeycoin';
import { EasyPay } from '@fundkit/easypay';

// Initialize providers
const honeycoin = new HoneyCoin({
  apiKey: process.env.HONEYCOIN_API_KEY!,
  publicKey: process.env.HONEYCOIN_PUBLIC_KEY!,
});

const easypay = new EasyPay({
  apiKey: process.env.EASYPAY_SECRET!,
  clientId: process.env.EASYPAY_CLIENT_ID!,
});

// Create payment client
const client = new PaymentClient({
  apiKey: process.env.FUNDKIT_API_KEY!,
  providers: [honeycoin, easypay],
  environment: 'sandbox', // Use 'production' for live transactions
});

export default client;
```

```javascript JavaScript
const { PaymentClient } = require('@fundkit/core');
const { HoneyCoin } = require('@fundkit/honeycoin');
const { EasyPay } = require('@fundkit/easypay');

// Initialize providers
const honeycoin = new HoneyCoin({
  apiKey: process.env.HONEYCOIN_API_KEY,
  publicKey: process.env.HONEYCOIN_PUBLIC_KEY,
});

const easypay = new EasyPay({
  apiKey: process.env.EASYPAY_SECRET,
  clientId: process.env.EASYPAY_CLIENT_ID,
});

// Create payment client
const client = new PaymentClient({
  apiKey: process.env.FUNDKIT_API_KEY,
  providers: [honeycoin, easypay],
  environment: 'sandbox',
});

module.exports = client;
```

</CodeGroup>

## Step 4: Process Your First Payment

<CodeGroup>

```typescript TypeScript
import client from './payment';

async function processPayment() {
  try {
    // Create a transaction
    const transaction = {
      amount: 5000, // Amount in smallest currency unit (e.g., cents)
      currency: 'UGX',
      operator: 'mtn', // mtn, airtel, etc.
      accountNumber: '256779280949',
      externalId: 'tx_' + Date.now(),
      reason: 'Test payment from FundKit'
    };

    // Process the payment
    console.log('Processing payment...');
    const response = await client.collection(transaction);
    
    console.log('Payment initiated:', {
      provider: response.provider,
      transactionId: response.data.transactionId,
      status: response.data.status
    });

    // Check payment status
    const status = await client.getTransaction({
      provider: response.provider,
      txId: response.data.transactionId,
    });
    
    console.log('Payment status:', status);
    
  } catch (error) {
    console.error('Payment failed:', error);
  }
}

processPayment();
```

## Step 5: Run Your Integration

Set up your environment variables in a `.env` file:

```bash .env
FUNDKIT_API_KEY=fk_test_your_sandbox_key
HONEYCOIN_API_KEY=your_honeycoin_key
HONEYCOIN_PUBLIC_KEY=your_honeycoin_public_key
EASYPAY_SECRET=your_easypay_secret
EASYPAY_CLIENT_ID=your_easypay_client_id
```

Then run your code:

```bash
node payment.js
# or for TypeScript:
npx tsx payment.ts
```
